A desktop OS-inspired portfolio built with React + TypeScript. Features a boot sequence, draggable windows, a weather widget, and a music player — all running in the browser.

## Features

- 🖥️ Boot screen with BIOS animation
- 🪟 Draggable & resizable windows
- 🌤️ Live weather widget
- 🎵 Built-in radio player (Tunify)
- 📱 Mobile responsive layout

## Getting Started
```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Vite
- Zustand (state management)
- OpenWeatherMap API
